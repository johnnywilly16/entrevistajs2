Instruções

Você deverá desenvolver suas soluções no arquivo script.js. Durante as partes de Array, Loop e Objeto, você poderá observar o resultado das suas soluções via console. Ao chegar na parte de DOM, você poderá consultar o container de teste, localizado logo abaixo das instruções, para verificar se as alterações feitas tiveram o efeito esperado.

Passo a passo:

Array

  1. Crie um array de números 5 dezenas quaisquer ex: 55, 12, 30, 81,92
  2. Mostre no console o primeiro elemento
  3. Mostre no console o último elemento (dinâmicamente)

Loop

  1. Mostre dinamicamente no console todos os itens do array.

Objeto

  1. Crie um objeto casa com as seguintes propriedades:
    • tipo -> uma string com valor 'Casa térrea'
    • numero -> um numero com valor 125
    • moradores -> um array de duas posições, sendo a primeira uma string ' Fernanda ', e a segunda uma outra string 'José'
  2. Mostre no console o valor da propriedade numero do objeto criado
  3. Mostre no console apenas o primeiro nome da propriedade "moradores".

DOM

Observe a estrutura do arquivo template.html para desenvolver o que é pedido. Você não deve alterar o arquivo html diretamente, mas sim utilizando DOM no arquivo script.js .

  1. Selecione o elemento cuja classe é tech-list
  2. Crie um novo elemento li, utilizando DOM.
  3. Modifique o texto do elemento criado para 'Javascript'
  4. Adicione o elemento ao fim da lista cuja classe é tech-list
